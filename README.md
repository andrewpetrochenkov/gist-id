### Installation
```bash
$ [sudo] pip install gist-id
```

```bash
$ [sudo] npm i -g gist-id
```

#### Examples
```bash
$ cd path/to/gist
$ gist-id .
<id>
```

```bash
$ find ~/git/gists -type d -mindepth 1 -maxdepth 1 -exec gist-id {} \;
```
